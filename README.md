# My Portfolio

## Setup

```bash
pnpm install
pnpm dlx shadcn init
pnpm dlx shadcn add @skiper-ui/skiper40
```

## Add more Skiper UI components

```bash
pnpm dlx shadcn add @skiper-ui/skiper26  # Theme toggle
pnpm dlx shadcn add @skiper-ui/skiper4   # Theme toggle buttons
pnpm dlx shadcn add @skiper-ui/skiper41  # Card carousel
```

## Dev

```bash
pnpm run dev
```
